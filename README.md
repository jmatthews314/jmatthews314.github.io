# Personal Portfolio Website

This repository contains a simple portfolio website built for GitHub Pages.

## Setup Instructions

1. Create a repository named `yourusername.github.io`.
2. Upload `index.html` to the repository.
3. (Optional) Upload `resume.pdf`.
4. Edit the placeholder text with your information.
5. Visit `https://yourusername.github.io`.

## Customization

Update:
- Your name
- LinkedIn URL
- GitHub URL
- Email
- Project descriptions

## Suggested Additions

- `resume.pdf`
- `images/` folder for screenshots
- Additional project cards
